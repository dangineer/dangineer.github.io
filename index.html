Practice Questions

Just between us, I've shared the list of questions I ask my candidates. These questions are targeted for interns and full time candidates of various experiences so bear that in mind:) Difficulty starts from Caro-Taro (easiest), Car-Swizzlin (eh, bit more thinkin') to Zhangster-Swag (woah, slow down ther' horsey) You can either send me the solution via email (next button) or send a link to a hackerrank codepair with your solution~ Happy coding!

Caro-Taro


1. Write a function that tests whether a string is a palindrome. https://en.wikipedia.org/wiki/Palindrome

2. Check if two strings are anagrams. https://en.wikipedia.org/wiki/Anagram

3. Find the first non-repeated character (or caro-ter) in a string. Example: string = "aaaab", answer="b"

4. Write a function to compute the nth Fibonacci number.

Car-Swizzlin


1. Find the second highest integer in an unsorted array.

2. Write a program to calculate the nth prime number.

3. Rotate an array of n elements to the right by k steps. For example, with n = 7 and k = 3, the array [1,2,3,4,5,6,7] is rotated to [5,6,7,1,2,3,4].

4. Given two sorted integer arrays A and B, merge B into A as one sorted array.

Zhangster-Swag

mostly asked to full-time candidates

1. Find all pairs in array of integers whose sum is equal to given number.

2. Find the depth of a binary tree.

3. Check to see if a tree is balanced.

Extra caro-edit: Given an unsorted array of integers, find the length of the longest consecutive elements sequence. For example, given [100, 4, 200, 1, 3, 2], the longest consecutive elements sequence should be [1, 2, 3, 4]. Its length is 4.
